# Lucra Components (Experimental)

These UI components are available but experimental and may change. Use in non-production or with caution.

## LucraFlowView
- Embed a Lucra flow inside your React Native UI.

## Buttons and Pills
- `LucraCreateContestButton`: Native button to trigger contest creation.
- `LucraProfilePill`: Profile pill UI.

## Feeds and Cards
- `LucraMiniPublicFeed`: Compact public feed component.
- `LucraRecommendedMatchup`: Recommended matchup UI.
- `LucraContestCard`: Contest card UI.

Notes:
- APIs and appearance may change; test before relying in production.
- Some components may require the SDK to be initialized and the user configured.
