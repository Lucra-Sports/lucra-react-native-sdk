# Games You Play (GYP)

Games You Play includes flows and headless APIs for creating and managing game matchups.

## Entry Points

- Full-screen flows for creating and viewing matchups.
- Headless APIs for creating and retrieving matchups.
- Event listeners for matchup lifecycle events.
