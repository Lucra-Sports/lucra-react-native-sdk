# GYP Headless Functions

These APIs let you create and fetch game matchups without displaying UI.

## Create a Matchup

```ts
const res = await LucraSDK.createGamesMatchup('DARTS', 1.0);
const matchupId = res.id;
```

Parameters:

- `gameId` (string): Your internal game identifier.
- `amount` (number): Wager amount in USD.

## Get a Matchup

```ts
const matchup = await LucraSDK.getGamesMatchup(matchupId);
```

Parameters:

- `matchupId` (string): Lucra matchup identifier.
