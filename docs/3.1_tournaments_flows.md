# Tournament Flows

[Lucra Flows setup](1.2.7_lucraflows.md)

Use `LucraSDK.present` to navigate tournament experiences.

## View Tournament Details

Use `MATCHUP_DETAILS` to show a tournament matchup:

```ts
await LucraSDK.present({
  name: LucraSDK.FLOW.MATCHUP_DETAILS,
  matchupId: 'TOURNAMENT_MATCHUP_ID',
});
```

Required params:

- `name`: `LucraSDK.FLOW.MATCHUP_DETAILS`
- `matchupId`: Tournament matchup ID

## Find/Create Available Tournaments

Use `CREATE_GAMES_MATCHUP` to surface available tournaments (same entry point used for games):

```ts
await LucraSDK.present({
  name: LucraSDK.FLOW.CREATE_GAMES_MATCHUP,
});
```

Required params:

- `name`: `LucraSDK.FLOW.CREATE_GAMES_MATCHUP`
