# Tournament Headless Functions

Use the headless APIs to list, fetch, and join tournaments.

## Get Recommended Tournaments

```ts
const tournaments = await LucraSDK.getRecomendedTournaments({
  includeClosed: true,
  limit: 50,
});
```

Parameters:

- `includeClosed` (boolean): Include completed tournaments.
- `limit` (number): Maximum results.

## Get Tournament Matchup

```ts
const tournament = await LucraSDK.tournamentsMatchup('TOURNAMENT_ID');
```

Parameters:

- `id` (string): Tournament matchup ID.

## Join Tournament

```ts
try {
  await LucraSDK.joinTournament('TOURNAMENT_ID');
} catch (e) {
  if (e.code === 'INSUFFICIENT_FUNDS') {
    console.error('User has insufficient funds to join tournament');
  } else {
    console.error(e);
  }
}
```

Possible errors:

- User not in a valid state to join
- Location services failure
- Compliance validation failure
- Insufficient funds
