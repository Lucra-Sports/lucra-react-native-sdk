# SYW Flows

[Lucra Flows setup](1.2.7_lucraflows.md)

Use `LucraSDK.present` to launch Sports You Watch flows.

## Create Sports Matchup

```ts
await LucraSDK.present({
  name: LucraSDK.FLOW.CREATE_SPORTS_MATCHUP,
});
```

Required params:

- `name`: `LucraSDK.FLOW.CREATE_SPORTS_MATCHUP` (no additional parameters)

## View Matchup Details

```ts
await LucraSDK.present({
  name: LucraSDK.FLOW.MATCHUP_DETAILS,
  matchupId: 'MATCHUP_ID',
});
```

Required params:

- `name`: `LucraSDK.FLOW.MATCHUP_DETAILS`
- `matchupId`: Lucra matchup ID

## Public Feed

```ts
await LucraSDK.present({
  name: LucraSDK.FLOW.PUBLIC_FEED,
});
```

Required params:

- `name`: `LucraSDK.FLOW.PUBLIC_FEED`
